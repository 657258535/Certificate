# Certificate - 交互式合格证设计器

一个简单易用的在线合格证设计工具，帮助用户快速创建和下载商品合格证。

## 📋 功能特点

### 核心功能
- **实时预览**：编辑内容时实时查看合格证效果
- **双面设计**：支持合格证正反面设计
- **自定义内容**：可编辑品牌、品名、等级、质检员、执行标准、安全类别、成分面料、产地、价格等信息
- **图片上传**：支持拖放或点击上传二维码图片和品牌Logo
- **数据保存**：自动保存表单数据到本地存储，刷新页面不丢失
- **一键下载**：支持同时下载合格证正反面为PNG图片
- **一键重置**：快速恢复默认设置

### 设计特性
- **响应式布局**：适配不同屏幕尺寸
- **直观的用户界面**：左侧编辑面板，右侧实时预览
- **洗涤图标**：内置标准洗涤方法图标
- **可翻转卡片**：点击翻转按钮查看正反面效果

## 🚀 快速开始

### 在线使用
直接打开 `index.html` 文件即可使用，无需安装任何依赖。

### 本地部署
1. 克隆或下载项目到本地
2. 使用任意Web浏览器打开 `index.html` 文件
3. 开始设计您的合格证

## 📝 使用指南

### 编辑合格证信息
1. 在左侧编辑面板中填写各项信息：
   - 基本信息：品牌、品名、等级、质检员等
   - 背面信息：品牌英文、宣传标语、二维码说明

### 上传图片
1. **二维码图片**：拖放或点击二维码区域上传自定义二维码
2. **品牌Logo**：拖放或点击Logo区域上传品牌Logo

### 查看效果
1. 在右侧预览区实时查看编辑效果
2. 点击「翻转卡片」按钮查看合格证背面

### 下载合格证
1. 点击「下载合格证」按钮
2. 系统将自动下载合格证正面和背面的PNG图片

### 重置表单
1. 点击「重置」按钮可恢复默认设置

## 🔧 自定义开发

### 修改默认值
在 `index.html` 文件中，找到表单元素的 `value` 属性即可修改默认值。

### 调整样式
所有样式都定义在 `index.html` 文件的 `<style>` 标签内，您可以根据需要调整：
- 卡片尺寸和布局
- 字体样式和大小
- 颜色方案
- 洗涤图标

### 扩展功能
您可以根据需求扩展以下功能：
- 添加更多自定义字段
- 支持更多图片格式
- 实现批量生成功能
- 添加模板选择功能
